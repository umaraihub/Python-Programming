## 💡 Key Concepts at a Glance
```python
# Variables & Data Types
name    = "Muhammad Umar"   # str
age     = 26                # int
gpa     = 3.4               # float
active  = True              # bool

# f-String
print(f"My name is {name} and I am {age} years old.")

# Type Casting
marks = int("95")           # string → int
price = float(100)          # int → float

# Operators
percentage = (432 / 500) * 100   # arithmetic
print(percentage >= 50)          # comparison → True
print(percentage > 40 and percentage < 100)  # logical → True
```
